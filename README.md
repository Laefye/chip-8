# CHIP-8 Emulator

![Build Status](https://github.com/Laefye/CHIP-8/workflows/Build%20and%20Release/badge.svg)

## Requirements

- CMake 3.2 or higher
- C++20 compatible compiler
- SDL3 library

## Download

Pre-built Windows binaries are available:
- **Releases**: Download stable versions from the [Releases](https://github.com/Laefye/CHIP-8/releases) page

## Usage

```bash
./CHIP8Emu <path-to-rom>
```
